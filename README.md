# British Schools Asia

A curated directory of Asia's top British schools. Built with Astro, Tailwind CSS v4, and deployed on Cloudflare Pages.

**Domain**: britishschoolsasia.com

## What this is

A premium, editorially curated index of British curriculum schools across Asia's top cities. Schools must meet strict inclusion criteria: English National Curriculum, minimum USD 18,000 Year 3 fees, published fee schedules, and location in a top-100 Asian city.

Schools are ranked by fees with reputation-based adjustments derived from at least 3 independent sources.
